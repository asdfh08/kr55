# Тренажер слов - Express.js приложение

## Описание
Простое приложение для изучения иностранных слов на Express.js. Позволяет добавлять, удалять и практиковать слова.

## Установка и запуск
1. Установите Node.js
2. Скачайте файлы проекта
3. В папке проекта выполните данные команды:
```bash
npm install
node server.js
```
4. Откройте: `http://localhost:3000`

## API
- `GET /api/words` - все слова
- `GET /api/words/random` - случайное слово
- `POST /api/words` - добавить слово
- `DELETE /api/words/:id` - удалить слово
- `PUT /api/words/:id/learned` - изменить статус изучения

## Структура
- `server.js` - главный файл
- `routes.js` - маршруты
- `controller.js` - обработка данных
- `middleware.js` - логирование
- `words.json` - база слов
- `public/` - фронтенд

## Функции
1. Добавление новых слов
2. Практика случайных слов
3. Отметка изученных слов
4. Удаление слов
